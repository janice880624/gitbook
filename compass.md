# 指南針

### 電子羅盤校正

step1. 再**更多**的類別中找到**電子羅盤校準**積木

{% hint style="info" %}
如果第一次使用，建議先校正墊子羅盤
{% endhint %}

![](.gitbook/assets/image%20%2820%29.png)

step2. 儲存檔案，下載到 micro:bit 進行校準

step3. 校準時顯示 【DRAW A CIRCLE】 跑馬燈，請旋轉 micro:bit 話一個圓

![](.gitbook/assets/image%20%2825%29.png)

step4. 完成後會出現笑臉
